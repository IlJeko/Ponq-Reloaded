# Ponq Reloaded
 Project inspired by Atari's PS1 Ponq

 There are 3 different levels, the game returns in the menu after you or the opponent reach 5 points.

 The build file is in the root folder and is named Ponq Reloaded.exe


 # Controls:
 
 W/S - move paddle Up/Down
 
 Esc - Return to main menu
